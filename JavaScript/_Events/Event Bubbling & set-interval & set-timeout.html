        What is EventBubbling?

        ~ EventBubbling is the problem that is raised whenever we use nested html tags.
        ~ for ex:Consider a container that contains a child container and the child container also have child
        ~ whenever u want to perform the events on class child then autmatically the events are applied to the childContainer and the container  

        <div class="container">
            <div class="childContainer">
                <div class="child">Hey I am box
            </div>
        </div> 

        The only solution is to use the stopPropogation() method to overcome the bubbling.
